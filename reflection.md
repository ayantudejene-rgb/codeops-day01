1. What is the difference between Git and GitHub? 
git is a version control in our computer which we can track changes to our files. GitHub is a website that we can protect our codes if anything happens on our computer which we can backup our work and access it from anywhere.
2. What is the purpose of the terminal?
It is an interaction way with our computer instead of clicking through folders and icons we can type command to filesystem.
3. What does git add do?
It moves a file from the working directory into the staging area.
4. What is the purpose of git commit?
It takes everything in the staging area and saves it as a permanent snapshot in the project history.
5. Write the daily Git workflow in the correct order
1. git pull
2. git status
3. git add
4. git commit -m 
5. git push
